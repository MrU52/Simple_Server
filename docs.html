Simple API Server

Server URL: http://localhost:3000

What is this?

A simple web server that responds to different web addresses with JSON data.

Technologies Used

  • Node.js - Runs JavaScript on the server
  • Express.js - Makes creating web servers easy

Available URLs

GET /
Home page - Shows all available links
GET /health
Health check - Shows if server is working
Returns: {"status": "ok"}
GET /time
Current time - Shows current date and time
Returns: {"time": "2025-08-21T..."}
GET /json
Sample data - Shows example JSON data
Returns: {"data": [{"id": 1, "name": "test"}]}
GET /echo
Echo instructions - Explains how to send data
Returns: {"message": "Send POST to echo data"}
POST /echo
Echo back data - Sends back whatever you send to it
Send: Any JSON data
Returns: {"echo": your_data_here}

How to Start

  1. Open terminal/command prompt
  2. Go to the project folder
  3. Type: node server.js
  4. Visit: http://localhost:3000

How to Test

  • Easy way: Click any link in your browser
  • All URLs work: Just type them in your browser address bar
  • POST testing: Use tools like Postman or curl
